YT2Audi: YouTube for Audi
========================

YT2Audi is a powerful, cross-platform YouTube downloader and converter specifically optimized for Audi Q5 MMI, MIB2, and MIB3 infotainment systems. It automates the entire process from downloading to FAT32-compatible splitting and USB transfer.

Features
--------

* **High Performance**: Concurrent download/conversion pipeline with GPU acceleration (NVENC/AMF).
* **MMI Optimized**: Automatic resolution scaling and codec selection for flawless playback on Audi screens.
* **Smart Transfer**: Automatic USB drive detection and copying.
* **Robust Logic**: Resume support for long videos and persistent download history.
* **Auto-Splitting**: Slices large files to fit within FAT32 4GB limits while maintaining playback order.
